# Technology that drives fun cooking 2012
# 料理を支える技術 2012

本トークでは、クックパッドが具体的にどのような技術を組み合わせてサービスを構築しているのか、
と言った話から、 Rails 2.3 から Rails 3 への移行などの運用話、
コードのメンテナンスはどう行っているか、等の苦労話やノウハウをお話しいたします。
